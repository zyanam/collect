1. 自增变量，写出程序运行结果。

   ```java
       /**
        * 自增变量
        *
        * @param args
        */
       public static void main(String[] args) {
           int i = 1;
           i = i++;
           int j = i++;
           int k = i + ++i * i++;
           
           System.out.println("i=" + i);
           System.out.println("j=" + j);
           System.out.println("k=" + k);
       }
   ```
